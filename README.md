# xhtml.js
常用的dom操作集合。Common set of DOM operations.

****
### 作者:心叶
### 邮箱:yelloxing@gmail.com

如何使用？
--------------------------------------
```bash
npm install --save xhtml.js
```

安装好了，可以这样调用：

```js
import xhtml from 'xhtml.js';
```
